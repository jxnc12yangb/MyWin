　1，你想以40km/h的速度开车到80岁，还是以80km/h的速度开车到40岁？ ;

　　2，在敌人面前，谁先镇定下来，谁就离胜利不远了。 ;

　　3，所有的人都是平凡的，有些人因知道这一点而真正成了平凡的人。 ;

　　4，乞丐就是一种向你的良心征税的人。 ;　　5，所谓大难不死，就是有了灾祸之后，得到的不是同情，而是莫名其妙的祝贺。 ; 

　　6，处在社交圈中是一种烦恼，而超脱出来简直是一场悲剧。 ;

　　7，厌倦，就是一个人吃完盘子里的食物后对盘子的感情。 ;　　8，知识是一种使求知者吃得越多越觉得饿的粮食。 ;

　　9，羞耻心就像人的内衣，必要时脱掉了没什么，关键是为谁脱掉。 ;

　　10，爱情是会沉底的，在平淡的日子里，最重要的是经常轻轻晃一晃盛装爱情的“水杯”。 ;　　11，战争：用舌头解不开就用牙齿咬吧！ ; 

　　12，憎恨别人就像为了逮住一只耗子而不惜烧毁你自己的房子。但耗子不一定逮到了。 ; 

　　13，每个人的一生都有许多梦想，但如果其中一个不断搅扰着你，剩下的就仅仅是行动了。 ;

　　14，世上只有想不通的人，没有走不通的路。; 

　　15，幸运之神的降临，往往只是因为你多看了一眼，多想了一下，多走了一步。 ; 

　　16，失败发生在彻底的放弃之后。 ; 

　　17，所谓敌人，不过是那些迫使我们自己变得强大的人。 ; 

　　18，胆怯的人在危险前被吓住了；懦弱的人在危险中被吓住了；勇敢的人在危险过后被吓住了。 ; 

　　19，反腐败的风险常常大于搞腐败的风险。――人大代表的不平。 ;

　　20，使人疲惫的不是远方的高山，而是鞋里的一粒沙子。 ;

　　21，家庭的幸福需夫妻共同努力，而破坏它，一人就够了。 ; 

　　22，最了解你的人有时不是你的朋友，而是你的敌人。 ;

　　23，你若不想做，会找一个或无数个借口；你若想做，会想一个或无数个办法。 ; 

　　24，看一个人的心术，看他的眼神；看一个人的身价，看他的对手；看一个人的底牌，看他的朋友。
